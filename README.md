A mobile application built using Flutter that displays different categories of electronic products such as phones, laptops, gaming PCs, and TVs.

## Features
- Browse product categories
- View items with images
- Simple and clean UI design
- Multiple category screens

## Technologies
- Flutter (Dart)
- Firebase
- Material UI


## How to Run
1. Clone the repository
2. Run `flutter pub get`
3. Run the app using emulator or device
